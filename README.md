**Site statique "Emploi du temps"**

**Description**  
Site web statique présentant un emploi du temps (planning) en **HTML** et **CSS**. Simple, responsive et facilement personnalisable pour étudiants, enseignants ou équipes.

**Fonctionnalités**
- **Affichage** d'un emploi du temps hebdomadaire
- **Responsive** (bureau, tablette, mobile)
- **HTML** simple + **CSS** modulaire
- Ajout manuel d'événements dans le HTML
- Styles faciles à personnaliser (couleurs, polices)

**Prérequis**
- Navigateur moderne (Chrome, Firefox, Edge, Safari)  
- Aucun serveur requis — fonctionne en local (double-clic sur `index.html`)

**Installation / utilisation**
1. Cloner le dépôt ou télécharger les fichiers :  
   `git clone <URL-du-dépôt>`
2. Ouvrir `index.html` dans votre navigateur.
3. Modifier les créneaux dans `index.html` (balises `<div>` ou `<td>` selon la structure).
4. Personnaliser `styles.css`.

**Structure du projet**
- `index.html` — page principale  
- `styles.css` — styles CSS  
- `assets/` — images, polices, icônes (optionnel)  
- `README.md` — documentation

**Exemple rapide**
- Ouvrir `index.html`  
- Copier un bloc de cellule pour ajouter un créneau  
- Modifier texte/heures, sauvegarder et rafraîchir

**Personnalisation recommandée**
- Utiliser **CSS Grid** ou **Flexbox** pour la mise en page  
- Définir des variables CSS dans `:root` pour couleurs/thème  
- Optionnel : ajouter un petit script JavaScript pour import/export JSON des événements
